To view a WebTrends report as an Excel document, install the "WebTrends Document Utility" program.  
Be sure you have the latest version installed on your machine.

Insure Latest [UT]

Note: You may skip to step 3 if you have previously completed steps 1 and 2 with the latest version.
  1. Download latest version "WebTrends Document Utility 4.0"
  2. Install documentutilityinstall.exe by double clicking the file in the directory where is was saved.  
    (After the installation you may need to close all browser windows and re-open your browser.)

    Example C:/Program Files/WebTrends/documentutilityinstall.exe

  3. Convert into a Microsoft Excel document.

    When prompted by the browser, select the option "Open it".

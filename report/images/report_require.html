WebTrends Enterprise Reporting Server
 
In order to use the WebTrends Enterprise Reporting Server Interface you will need to use either

Microsoft Internet Explorer version 4.0 or higher OR Netscape version 4.04 or higher.
 
Note: Please be sure that your browser has Javascript enabled.
  • In Netscape: "Preferences..." dialog
    • Advanced | Enable Javascript

  • In Internet Explorer: "Internet Options" dialog.
    • Security
      • Settings | Enable Active scripting
